2、组件的三大属性（props,refs,props)

一：state的三个状态：初始化，更新，渲染

注意：

  • this绑定的方法：

​ 1、return <h2 onClick={this.handleClick.bind(this)}>{islike?'A':'B'}</h2>//问题是效率低，每次渲染都要渲染一次该方法。

​ 2、this.handleClick=this.handleClick.bind(this);//效率高，常用写法，不是最好，箭头函数最好

以上两种方法的原理都是一样，就是创建一个新的方法this.handleClick.bind()，而内容就是handleClick内部的内容。

​ 3、箭头函数效果最好，handleClick=()=>{..}即可。

  • 想要改变state的状态const islike=!this.state.islike;，但是如果需要重新渲染页面显示一样的值就需要使用this.setState(）来更新页面了。

三：refs属性：（标识组件的某个元素）

app.js:

注意：

  • 第一种设置ref的方法（不推荐）：<input ref={input=>this.input=input} />

    获取dom对象：const input = this.refs.content即可以获取input

  • 第二种ref设置为回调函数形式（推荐）：<input ref={input=>this.input1=input} />回调函数的第 一个 input是对应的标签名。而this.input1即为添加到组件对象上的一个属性，名字可以随便起获取：this.input1.value即可以获取，即可以获取input的value属性。

  • 失焦事件为：onBlur={this.handleBlur}，使用为调用event上的一个target即可获取当前的input对象。注意的是，该this指向必须指向为组件对象的前提下，否则event的获取不正确。
