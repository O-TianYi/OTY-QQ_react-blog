react-router4（react路由）

一、区别：

router和route的区别：

  • router路由器，router.get(path,fun(req,res){})，该方法为后台路由的注册路由方法，注册一个key为path,value是后面的回调函数的一个路由。path就是你在地址栏上的路径。前端路由的注册路由为：。

  • route是路由，一个路由就是一个映射关系，即key:value，key为路由路径，value分为两种，如果为后台路由即第二个参数为回调函数，如果是前端路由即是一个组件。

路由的基础原理：（history）

就是浏览器的自身携带的一个history的方法，以及他的一些方法：（react使用了一个）

history.push(path)：添加一个地址，用于记忆地址变化。

history.goBack(path)：回退前一个地址。

history.goForward(path)：前进一个地址。

history.replace(path)：替换当前地址。记住是替换。

注意：可以象限为histroy是一个数组，理解就差不多了。
