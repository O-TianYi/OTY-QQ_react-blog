使用js实现react路由的跳转功能

使用withRouter

例如：实现注册页面，注册完成之后自动跳转到登陆页面的功能：

注意：

（1）只需要在react-router-dom上引入withRouter，然后就可以使用react-router的三个对象history, location, match，，实现用js跳转路由的功能。

（2）调用history实现跳转：this.props.history.push(‘/login’)。

histroy的方法有：

​ history.push(path)：添加一个地址，用于记忆地址变化。

​ history.goBack(path)：回退前一个地址。

​ history.goForward(path)：前进一个地址。

​ history.replace(path)：替换当前地址。记住是替换。
